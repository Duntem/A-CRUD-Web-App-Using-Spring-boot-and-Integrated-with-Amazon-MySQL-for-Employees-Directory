## 𝑪𝑹𝑼𝑫 𝑺𝒑𝒓𝒊𝒏𝒈 𝑩𝒐𝒐𝒕 𝑬𝒎𝒑𝒍𝒐𝒚𝒆𝒆 𝑫𝒊𝒓𝒆𝒄𝒕𝒐𝒓𝒚 𝑨𝒑𝒑𝒍𝒊𝒄𝒂𝒕𝒊𝒐𝒏. 
I designed and developed a full-stack web application that performs Create, Read, Update, and Delete (CRUD) operations for an employee directory. 
The application was built using Spring Boot, Java, and IntelliJ IDEA, with backend data stored in Amazon RDS. 
Testing and API validation were carried out using Postman, while Sqlectron was used for database interaction and monitoring.
